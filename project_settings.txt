Сборка:
    + Этап очистки:
	Команда:		    echo
	Параметры:		    "$(cat %{sourceDir}/../pwd)" | sudo -S bash %{sourceDir}/cleaning
Запуск:
    + Этап развёртывания:
	Команда:		    echo
	Параметры:		    "$(cat %{sourceDir}/../pwd)" | sudo -S bash %{sourceDir}/deployment %{ActiveProject:RunConfig:Executable:Path} %{sourceDir}
    Запуск:
	Параметры командной строки: -r

В родительском относительно проекта каталоге ожидается текстовой файл pwd с паролем без переноса строки для sudo.
При сборке проект полностью инсталлируется в систему, за исключением файлов лицензии и лога изменений.
При очистке - аналогичная деинсталляция.
